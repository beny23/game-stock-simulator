# Stock Camp Simulator

Offline, GM-led stock market simulation for scout camp. Built with Phaser 3.

## Dev

- Install: `npm install`
- Run: `npm run dev`

## Build (offline)

- Build: `npm run build`
- The `dist/` folder is a static site you can open offline (or serve locally).
